| arrayToSearch values       | 1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 | 9 |
| elementsToSearchFor values |   | 2 |   |   | 5 |   |   | 8 |   |
| Sum = 3                    | 0 | 1 | 0 | 0 | 1 | 0 | 0 | 1 | 0 |
